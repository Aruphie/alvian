ABOUT ME

Halo semuanya! perkenalkan nama saya Alvian Raihan Ramadan saya saat ini sedang menempuh pendidikan di salah satu perguruan tinggi negeri di Indonesia yaitu IPB penjurusan yang saya ambil yaitu Fisika. Sejak kecil saya sudah memiliki minat dengan pelajaran yang berbau hitungan dan problem solving seperti matematika dan fisika tapi karena seiring berjalanya waktu saya merasa saya lebih tertarik untuk mendalami fisika maka saya mengambil penjurusan Fisika di kampus saya. Berhubungan dengan problem solving saya juga berminat di dunia pemrograman di mana dalam dunia pemrograman problem solving itu hal yang diutamakan karena dalam dunianya sendiri pemrograman adalah sebuah cara untuk menyelesaikan masalah jadi pasti di dalamnya perlu skill problem solving yang baik untuk memecahkan masalah tersebut.


Untuk memulai belajar bahasa pemrograman saya mulai dari mencari-cari tips belajar pemrograman untuk pemula di internet dan saya menemukan bahwa untuk memulai belajar alangkah baiknya dimulai dengan bahasa Python, dan sampai sekarang saya masih mendalami bahasa python ini karena ternyata python itu sendiri sangat luas cakupannya dan sepertinya tidak masuk akal untuk menguasai semuanya dalam waktu pendek. Perlu waktu yang panjang untuk setidaknya lancar memprogram pyhton dan librarinya. oleh karena itu saya akan menyimpan dokumentasi project saya disini dengan maksud agar ketika saya melamar kerja saya bisa memberikan saja web ini kepada HRD untuk bahan pertimbangan, selain itu juga saya ingin berbagi pengalaman saya selama saya belajar dan mendalami python ini.
